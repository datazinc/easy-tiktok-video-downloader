FAQ:

Can you give videos without watermarks?

No. This tool is built to only use information provided to your browser from TikTok. Non-watermark links can only be obtained(AFAIK) using a server-side solution. If there is an easy way to retrieve it without the use of a third party service, send me a feature request with instructions.
